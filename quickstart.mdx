## Step 1: Register Your Account

Start your CloudThinker journey by creating your account.

<Card title="Sign Up Now" icon="user-plus" href="https://app.cloudthinker.io/register">
  Create your free CloudThinker account
</Card>

### Registration Process

1. **Visit Registration Page**: Go to [app.cloudthinker.io](https://app.cloudthinker.io)
2. **Enter Your Details**:
   - Full Name
   - Email Address
   - Create a secure password
3. **Verify Your Email**: Check your inbox and click the verification link
4. **Complete Profile**: Add any additional profile information

<Note>
After registration, you'll be automatically redirected to the onboarding flow to set up your workspace.
</Note>

### Connection Setup

<Tabs>
  <Tab title="AWS Setup">
    **Required Credentials:**
    1. **Access Key ID**: Your AWS access key (starts with AKIA...)
    2. **Secret Access Key**: Your AWS secret access key
    
    **Security Best Practices:**
    - Use IAM users with minimal required permissions
    - Prefer read-only access keys when possible
    - Enable MFA for your AWS account
    
    <Warning>
    Your credentials are encrypted and stored securely using industry-standard encryption.
    </Warning>
  </Tab>
  
  <Tab title="GCP Setup">
    **Required Information:**
    1. **Service Account Key**: JSON key file from your GCP service account
    2. **Project ID**: Your Google Cloud project identifier
    
    **Setup Instructions:**
    - Create a service account in GCP Console
    - Download the JSON key file
    - Grant minimal required permissions
  </Tab>
  
  <Tab title="Azure Setup">
    **Required Credentials:**
    1. **Client ID**: Azure application client ID  
    2. **Client Secret**: Azure application secret
    3. **Tenant ID**: Your Azure AD tenant ID
    4. **Subscription ID**: Target Azure subscription
    
    **Prerequisites:**
    - Register an application in Azure AD
    - Grant appropriate RBAC permissions
    - Generate client secret
  </Tab>
  
  <Tab title="Sandbox">
    **No Setup Required!**
    
    Sandbox mode lets you:
    - Explore CloudThinker's interface
    - See how AI agents work
    - Try operation templates
    - Learn the platform safely
    
    Perfect for evaluation and learning before connecting real infrastructure.
  </Tab>
</Tabs>
