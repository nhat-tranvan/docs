# CloudThinker Landing Assets

Generate on-brand feature card illustrations for CloudThinker's landing page using Nano Banana Pro (Gemini image gen).

**Style:** Dark Technical Blueprint Schematic — developer architectural diagrams on dark charcoal backgrounds featuring dashed offset wireframes, solid colored strokes, dark tinted fills, and intersection nodes. Zero glow, purely flat vector.

## Workflow

1. Receive feature card request (feature name + description)
2. Build prompt by combining the **Style Guide** below with the specific content
3. Generate image via Nano Banana Pro script
4. Deliver the result

## Model Selection

Two image gen models available. Choose based on need:

| Model | ID | Speed | Quality | Use When |
|-------|----|-------|---------|----------|
| **Nano Banana Pro** | `gemini-3-pro-image-preview` | ~24s | Higher detail, larger files | Final assets, hero images |
| **Nano Banana 2** | `gemini-3.1-flash-image-preview` | ~10s (2.3x faster) | Good, lighter files | Iterations, drafts, bulk gen |

**Default: Nano Banana 2** (fast iterations). Switch to Pro for final polish.

## Generation Command

**Generation Command (Prompt Template):**
```text
A Dark Technical Blueprint / Schematic illustration for a SaaS landing page. 

CRITICAL STYLE RULES (ALWAYS DARK MODE):
- BACKGROUND: MUST BE Pure deep black (#050505 or #000000). NEVER use white or light backgrounds. NO gradients.
- FLAT VECTOR ONLY: Absolutely NO 3D, NO glows, NO neon, NO shading.
- COLOR PALETTE: Only TWO colors used throughout. 1) Medium Gray for dashed lines and text. 2) ONE single Accent Color (e.g., Teal, Blue, or Amber) for main shapes.
- SHAPE CONSTRUCTION (THE 'BLUEPRINT' LOOK):
  - Primary Shapes: Drawn with a solid Accent Color stroke, and filled with a very dark tint of that Accent Color (so it looks flat but slightly lighter than the pure black background). NEVER use white/light gray fills.
  - Offset Dashed Wireframes: Main shapes MUST be surrounded by offset Dashed Gray outline paths. 
  - Icons/Details: Drawn using Accent Color strokes inside the main shapes or dashed boxes.
  - Nodes: Small solid circular dots in the Accent Color placed at the corners of dashed outlines or at line ends.
- CONNECTIONS: Dashed gray lines connecting the elements.

SCENE COMPOSITION:
- Extremely minimalist, structural, architectural diagram aesthetic.
- Text: Small, gray monospace or clean sans-serif.
ASPECT RATIO: 16:9
```

## Prompt Construction

Combine style guide + specific content + accent color. Example:

**Request:** "Create Incident Response card illustration"

**Full prompt:** `<style guide above> The subject is 'Incident Response'. Show a triangle with exclamation mark (alert symbol) in coral/red (#E85A6F), centered. Surround it with dotted circular orbit paths with small coral dots at cardinal points (N, E, S, W). Below the triangle, a horizontal dashed timeline/progress bar with small vertical tick markers. Monospace text label: 'MTTR: 4m 32s' in muted gray. Faint dashed concentric diamond shapes behind the triangle. All structural lines in dark gray (#3A3A50), accent elements in coral (#E85A6F) only.`

## Tips

- Keep illustrations SPARSE — less is more. The dark background does the heavy lifting
- Only ONE accent color per card. Secondary elements stay in dark gray
- Dashed lines are the signature — use them everywhere
- Small colored dots at intersection/connection points add the "live system" feel
- Monospace labels should feel like terminal output
- The style should feel like looking at a submarine sonar screen or NORAD terminal
