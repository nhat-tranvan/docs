## Quick Setup

<Steps>
  <Step title="Install CloudThinker App">
    Go to **Settings > Integrations > Slack** in CloudThinker and click **"Add to Slack"**
  </Step>
  <Step title="Authorize Permissions">
    Grant CloudThinker access to:
    - Send messages
    - Read channel information  
    - Access user mentions
  </Step>
  <Step title="Invite to Channels">
    Add the CloudThinker app to your desired Slack channels
  </Step>
  <Step title="Test Connection">
    Try: `@CloudThinker show AWS costs` to verify everything works
  </Step>
</Steps>

## Basic Commands

### Talk to CloudThinker

Simply mention @CloudThinker in your Slack channels and specify which agent or operation you need:

```bash
# Cost analysis
@CloudThinker alex show this month's AWS spending
@CloudThinker alex find cost optimization opportunities

# Security checks  
@CloudThinker oliver scan production environment for vulnerabilities
@CloudThinker oliver show compliance status

# Database performance
@CloudThinker tony analyze slow queries on production database
@CloudThinker tony check backup status

# Kubernetes operations
@CloudThinker kai show EKS cluster health  
@CloudThinker kai optimize pod resource allocation

# Strategic insights
@CloudThinker anna provide infrastructure status summary
@CloudThinker anna coordinate incident response
```

## Troubleshooting

<Accordion title="Agents not responding">
**Quick fixes:**
- Check if CloudThinker app is added to the channel
- Verify you're using @ mentions correctly (@CloudThinker alex, not alex)
- Try `/cloudthinker status` to check connection
- Make sure agents are configured in CloudThinker console
</Accordion>

<Accordion title="Missing alerts">
**Quick fixes:**
- Check alert configuration in CloudThinker Settings
- Verify Slack channel notification settings
- Test alerts with `/cloudthinker alerts test`
- Confirm alert thresholds are set correctly
</Accordion>

<Accordion title="Permission errors">
**Quick fixes:**
- Re-install CloudThinker Slack app
- Check workspace admin permissions
- Verify CloudThinker app has required Slack permissions
- Try removing and re-adding app to channels
</Accordion>
