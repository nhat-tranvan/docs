## Basic Usage

### Create Dashboards
```bash
@alex #dashboard monthly cost analysis
@tony #dashboard database performance 
@kai #dashboard cluster health
@oliver #dashboard security status
@anna #dashboard infrastructure overview
```

### Generate Reports
```bash
@alex #report quarterly cost summary
@oliver #report security compliance
@tony #report database optimization
@kai #report container performance
@anna #report executive summary
```

### Beta Features
```bash
# Presentation slides
@anna #slide Q1 infrastructure review
@alex #slide cost optimization results

# Websites and portals
@kai #web service status page
@oliver #web security dashboard
```
